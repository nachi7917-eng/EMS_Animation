# Earth–Moon–Sun Animation

A scientific animation created using Manim that visualizes the Earth–Moon–Sun system.

## Features

- Earth's revolution around the Sun
- Moon's revolution around Earth
- Orbital paths

## Technologies

- Python
- Manim
- Git
- GitHub

- Seasons
- Eclipses
- Moon phases
